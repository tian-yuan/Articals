#### windows 编译 folly

##### 准备条件

* vs2015
* vs2015 需要安装 window 10 sdk，在控制面板，点击修改，update vs 2015 选择 windows 10 sdk
* 编译最好在英文环境，vs 更新语言包，否则会出现一些列跟中文有关的问题
* vs2015 只支持一部分 c++14，所以可能会出现一些 c++14 相关的编译问题

> 目前最新版本是 此c++14，最好使用 vs2017 来编译
