注意:

* BSS 段

```
static int    local_C = 0;        //BSS段  
static int    local_D;            //BSS段
char        *p1;                     //BSS段  

1、未初始化的全局变量
2、初始化为 0 的静态变量
3、未初始化的静态变量
```

* 数据段

```
int          g_B       = 20;         //数据段  
static int   g_C       = 30;         //数据段  

初始化的全局变量
初始化不为 0 的静态变量
```

* 代码段

```
const int    g_A       = 10;         //代码段  
全局初始化 const 变量，只读的
代码段只保存只读数据，包括代码和 全局或局部 const 变量
```
